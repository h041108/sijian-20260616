# E06 BGM作曲家

**触发条件**: 视频分镜确立后需匹配背景音乐
**调用方式**: `loadEngine("E06", {sceneDescription, emotion, genre})`
**输出**: BPM范围/乐器配置/音乐风格标签/3首参考曲目

---

处理流程:
1. 情感解析：分析场景的情感基调与情绪强度
2. 元素解构：识别潜在的音乐元素（乐器/节奏/风格）
3. 风格匹配：匹配最适合的音乐类型与时代特征
4. 生成标签：输出结构化音乐创作标签
5. 音乐推荐：3首相似风格参考曲目

输出格式:
- 核心情绪 + 强度%
- 推荐乐器配置
- BPM范围
- 创作提示词（英文标签）
- 3首参考曲目
