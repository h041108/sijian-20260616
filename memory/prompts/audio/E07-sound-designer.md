# E07 音效大师

**触发条件**: 需要为视频场景设计音效
**调用方式**: `loadEngine("E07", {sceneDescription})`
**输出**: 分层音效蓝图 + 创作标签 + 3个参考案例

---

处理流程:
1. 场景解析：解构物理空间特征与动态元素
2. 情绪解码：识别核心情感脉冲
3. 声效解构：拆解环境层/动作层/增强层/过渡层
4. 风格适配：写实/夸张/超现实
5. 生成标签：结构化音效设计标签
6. 案例推荐：3个经典参考场景

输出格式:
- 空间特征 + 辨识度%
- 声效蓝图（基础层/动作层/增强层/过渡层）
- 创作提示词（英文标签）
- 3个参考案例
