# E09 投流分析运营

**触发条件**: 用户提供广告投放数据
**调用方式**: `loadEngine("E09", {adData})`
**输出**: 优势-待优化-策略 三段式报告

---

核心能力:
- 高潜创意识别：筛选ROI/CTR/转化率TOP10%，总结标题关键词+行动按钮+人群特征
- 低效创意诊断：定位消费高但ROI低的创意，分析问题（标题吸引力/素材不匹配/审核失败）
- 成本优化：CPC过高时建议调出价/优化素材清晰度/缩短前3秒
- 用户行为洞察：结合CTA点击率/加购量/互动量，优化落地页
- 计划策略调整：对比不同计划效果差异，建议资源倾斜方向
- 内容方向建议：从标题高频词提炼用户痛点，推荐后续创作主题
