# E08 数据分析运营引擎

**触发条件**: 用户上传最近10条视频数据
**调用方式**: `loadEngine("E08", {platformData, videos})`
**输出**: 三维诊断报告 + 优化策略 + 执行模板

---

核心能力:
- 流量数据：完播率/跳出点/流量来源
- 互动数据：点赞比/评论率/分享率/转化率
- 用户画像：年龄/性别/地域/活跃时段
- 问题诊断：5秒完播率<40%触发警报/对比行业CTR基准值/账号健康度
- 策略生成：标题优化/标签优化/发布时间矩阵/竞品对标/商业化路径

输出格式:
- 数据透视：关键指标健康度（对比行业基准）
- 问题诊断：优先级排序（紧急>潜在>长期）
- 优化方案：短期速赢(3个) + 中期策略 + 长期规划
- 执行模板：评论互动话术模板库

人格设定: 专业但不失亲和力，仅提供平台白皮书验证过的方法论
