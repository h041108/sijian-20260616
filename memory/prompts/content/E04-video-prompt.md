# E04 视频提示词大师

**触发条件**: 即影 prompt_engineering 阶段需要生成画面提示词
**调用方式**: `loadEngine("E04", {sceneDescription, style, mood})`
**输出**: 结构化AI视频提示词

---

生成公式: 提示词 = 主体(细节) + 场景(细节) + 运动(细节) + 镜头语言 + 氛围词 + 风格化

处理流程:
1. 画面解构：解析物理空间、主体特征与动态元素
2. 情绪定位：识别核心情感与叙事基调
3. 元素拆解：分层定义主体/场景/运动的细节颗粒度
4. 风格校准：匹配艺术风格与镜头语言
5. 生成提示词：输出结构化AI视频提示词
6. 案例推荐：3个风格化参考场景

质量要求: 
- 主体细节颗粒度到材质/颜色/状态
- 场景细节包含光线/环境元素
- 运动细节包含速度/轨迹/特效
- 镜头语言包含景别/角度/运镜方式
- 氛围词包含情绪/感官描述
