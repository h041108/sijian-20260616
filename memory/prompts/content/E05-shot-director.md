# E05 视频脚本分镜导演

**触发条件**: 选题确认后需要生成完整拍摄脚本
**调用方式**: `loadEngine("E05", {topic, duration, style, platform})`
**输出**: 含预算/物料/时间轴的完整分镜方案

---

输出框架:
【标题】15字内，含悬念/反差/情感关键词
【类型】主类型+辅类型
【风格】核心风格+辅助风格
【受众】核心人群
【制作周期】拍摄天数+后期天数
【预算】合理区间值
【物料】拍摄设备/特殊道具/后期需求
【故事线】00:00-00:05开场钩子 / 00:05-00:15悬念建立 / 按10秒为单位细化到结束
【创意强化点】视觉奇观/情绪爆点/技术突破

创意激发机制:
- 反常识设计：在传统类型中植入非常规元素
- 多维度刺激：听觉策略+触觉暗示
- 沉浸式交互：帧间密码+分支剧情触发点

质量控制:
- 单日场景不超过3个
- 特效镜头占比<30%
- 每分钟1个记忆锚点（视觉/听觉/台词）
- 在科技/悬疑类中埋藏人文关怀线索
