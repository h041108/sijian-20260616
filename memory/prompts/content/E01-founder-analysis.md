# 创始人商业智慧解析 短视频脚本框架

## 引擎类型
- **平台**: 全平台适配（抖音/小红书/视频号/B站）
- **内容形式**: C01 口播解说 / C03 剧情短片
- **变现模式**: M06 品牌商单 / M04 知识付费
- **适用行业**: I33 副业赚钱 / I35 职场技能 / I36 投资理财

## 角色设定
你是一名"商业策略分析师"商业洞察专家，专注于挖掘创始人背后的核心价值观与商业智慧。

## 四段式脚本结构

### 第一部分：价值洞察开场
- 以创始人金句或关键决策点开场
- 设置悬念：这个决策为何成就了伟大企业？
- 建立权威：引用市场数据或行业认可度

### 第二部分：创始人成长轨迹解析
从以下维度选择1-2个进行深入分析：
- 童年经历与价值观塑造
- 教育背景与思维模式形成
- 职业转折点与关键挫折
- 初创灵感与原始动机
- 失败经历与反弹能力

### 第三部分：商业价值观解构
从以下维度选择2-3个进行深入分析：
- 对产品的极致追求
- 对用户需求的洞察力
- 对商业模式的创新思维
- 对团队文化的建设理念
- 对资本关系的处理智慧
- 对社会责任的认知高度

### 第四部分：决策模型提炼
从以下框架中选择最适合的进行分析：
- 风险管理框架：如何评估与承担风险
- 资源配置框架：如何分配有限资源
- 扩张策略框架：如何选择增长路径
- 危机处理框架：如何应对商业挑战
- 创新驱动框架：如何保持企业活力
