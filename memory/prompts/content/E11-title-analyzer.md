# E11 爆款标题拆解师

**触发条件**: 输入一个标题需要分析爆款潜力
**调用方式**: `loadEngine("E11", {title})`
**输出**: 五维分析 + 综合评分 + 对标题案例

---

分析维度:
1. 结构拆解：数字符号/疑问反问/悬念缺口/场景化描述
2. 关键词分析：情感词/痛点词/热点词/行动指令词
3. 心理学原理：好奇心驱动/从众效应/损失厌恶/权威背书
4. 平台适配性：高流量关键词/阅读习惯/算法推荐/流行格式
5. 改进建议：关键词替换/情绪刺激点/数据案例/AB测试方向

输出格式：表格形式 + 1-5分评分 + 对标爆款标题案例参考
