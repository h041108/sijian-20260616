# E03 对标文案拆解引擎

**触发条件**: 用户提供对标账号内容链接或文案
**调用方式**: `loadEngine("E03", {targetContent, platform})`
**输出**: 结构化拆解报告

---

## 一、结构解构分析
- 框架层级识别：标注文案段落对应的模块类型（悬念开场/认知颠覆/方法论演示/情感共鸣/行动号召/价值升华）
- 分析模块排列逻辑（线性递进/平行对照/悬念嵌套）
- 节奏控制图谱：计算文字密度，每100字对应的信息量层级
- 标注转折点设计：识别认知转折词的位置与作用

## 二、爆款元素提取
- 认知冲突设计：找出刻意制造的3组认知落差（常识vs专业洞察 / 表面现象vs底层逻辑 / 传统解法vs创新策略）
- 权威构建手法：识别3类信任背书（数据背书/案例背书/逻辑背书）

## 三、创意手法解码
- 隐喻系统拆解：提取贯穿性隐喻，标注隐喻载体
- 画面感构建术：决策现场还原/数据具象化/概念实体化

## 四、延展创作方向
- 结构变体建议：前置结果法/案例嵌套法/悬念链设计
- 创意升级路径：反向推演版/时空折叠版/要素重组版

## 五、优化诊断报告
- 专业度增强：可插入专业符号位置、需增加决策链条展示环节
- 传播力优化：可转化互动梗、适合短视频的金句改写位置
