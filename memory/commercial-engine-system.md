## 零、核心架构

```
用户 → 选择行业 + 平台 + 变现模式
     → 引擎自动生成账号（模拟/API）
     → 引擎自动生成爆款内容（视频/图文）
     → 自动发布/排期
     → 数据追踪
     → 优化迭代
     → 用户看到收入
```

## 二、引擎组合器：平台 × 变现 × 内容 × 行业

每个引擎 = Platform + Monetization + Content + Industry + Prompt Profile

引擎命名规则：`{Platform}_{Monetization}_{Content}_{Industry}`

### 1000+ 引擎生成逻辑（自动排列组合）
```
引擎总数 = 10平台 × 10变现 × 12内容 × 40行业 = 48,000种

每个引擎包含：
- 账号搭建模板（头像/昵称/简介/背景图/第一个作品）
- 爆款选题库（100个标题/脚本模板）
- 内容生产流水线（DeepSeek脚本 + 即影视频 + 数字人口播）
- 发布排期表（最佳发布时间 + 频率）
- 数据追踪看板（播放/点赞/粉丝/转化/收入）
```

### 用户端体验（以 I01 护肤品带货 为例）

```
用户选择：小红书 + 带货 + 产品展示 + 护肤品
         ↓
引擎1：账号搭建
  - 昵称："XX成分党"
  - 简介："用数据说话，帮你找到最适合的护肤品"
  - 头像：即影生成（白底产品图+文字）
  - 背景：排版生成

引擎2：对标账号分析
  - 抓取前10名小红书护肤品带货博主
  - 拆解：选题策略 + 文案结构 + 转化话术
  - 输出可复制模板

引擎3：内容生产
  - 每日自动生成3条选题
  - 用户确认 → 即影自动生成视频
  - 费用：¥3/条（成本）+ ¥2（利润）= ¥5/条

引擎4：发布管理
  - 定时发布（7:30/12:00/18:30/21:00）
  - 自动添加标签和话题

引擎5：数据追踪
  - 实时播放量/点赞/收藏/评论
  - 达人协作：自动回复评论
  - 周报：哪个选题效果好
```
