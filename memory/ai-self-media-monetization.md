# AI + 自媒体变现知识库

## 工具矩阵
| 类别 | 工具 | 用途 |
|------|------|------|
| 文案 | DeepSeek | 选题、文案、脚本、标题 |
| 图片 | 即梦AI、可灵AI | 封面、配图 |
| 视频 | 即梦AI、可灵AI | 文生视频、图生视频 |
| 音频 | 海螺AI | 配音、BGM |
| 剪辑 | 剪映 | 合成、字幕、转场 |
| 自动化 | 扣子(Coze) + DeepSeek | 跨平台文案转换、自动分发 |
| 数字人 | DeepSeek + 数字人工具 | 口播视频、矩阵运营 |

## 各平台 AI 运营策略
- **小红书**：图文种草，娓娓道来语气。DeepSeek文+即梦图+自动标签。热门赛道：养生、语录、穿搭
- **抖音**：短平快，强钩子。数字人口播+矩阵。热门赛道：诗词动画、实体商家
- **视频号**：中年用户，信任驱动。养生/文化内容+私域裂变
- **B站**：深度内容，圈层文化。知识科普+二次元

## 变现路径（7条）
1. 广告收益 — 平台分成
2. 电商带货 — 橱窗/直播
3. 知识付费 — 课程/社群
4. 虚拟资料 — 引流售卖
5. 私域转化 — 微信高客单价
6. 代运营 — 为中小商家做AI内容
7. 矩阵账号 — 一人管10号

## 已验证案例数据
- 家居品牌：AI流水线 → 80-100篇/周（原15-20篇）
- 茶企：DeepSeek 45秒生成文案 + 混剪系统 15分钟千条视频 → 月销71斤
- 抖音→小红书：Coze+DeepSeek 1分12秒完成跨平台转换（比手动快20倍）

## 对思见的启示
- 即影 = 视频生成 + 数字人口播（已覆盖工具矩阵的视频/数字人两列）
- 缺失：跨平台分发、图文生成、自动化工作流
- 思见的爆款趋势拆解可以升级为"跨平台爆款拆解+一键复刻"
- 变现闭环：即影生成视频 → 自动生成多平台文案 → 用户发布 → 追踪数据 → 优化建议
