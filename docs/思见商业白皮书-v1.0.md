# 一、执行摘要

**思见是全球首个将人类思维过程实时可视化、可量化、可复用的认知操作系统。**

在大语言模型（LLM）将"知识回答"变为免费基础设施的时代，思见解决了LLM无法回答的一个核心问题：**"我是怎么想的？"** 

通过自研的61条思维线路引擎、四层意识识别系统、11种思维框架可视化技术，思见将人类隐性的思维过程转化为显性的、可分析的数据资产。面向教育机构、中小企业、家长三大市场，思见提供从个人思维训练到组织认知管理的全栈解决方案。

截至2026年6月，思见已建成完整的C端+B端产品矩阵，部署于Vercel全球边缘网络，并通过自研模型编排引擎实现多模型协同。

# 三、产品架构

## 3.1 C端：个人思维伙伴

```
用户输入 → 61条思维线路实时识别 → AI流式回复 + 思维线着色
                                → 11种框架可视化 → 思维空间
                                → 四层意识识别 → 认知日志
```

**核心功能：**
- 实时思维识别：输入框打字300ms即显示当前思维模式（因果线/对比线/发散线……）
- 11种框架空间：每轮对话自动选择最适合的思维拓扑可视化
- AI人格库：7个预设角色（数学导师、写作教练、批判性朋友……），一键切换思维模式
- 思维回顾卡片：对话结束后自动生成可截图分享的思维摘要
- 思维指纹：积累20条对话后生成个人专属四维雷达图

## 3.2 B端教育：思维训练基础设施

| 模块 | 功能 |
|------|------|
| 课堂热力图 | 教室实时看到每个学生的思维状态（跟上/走偏/卡住/超前） |
| 记忆宫殿 | 艾宾浩斯遗忘曲线驱动的概念记忆系统，300+种子题库 |
| 机构SaaS | 教师管理、学生管理、知识库、品牌定制、账单系统 |
| 学生成长对比 | 入训时 vs 1个月后 vs 3个月后的五维思维雷达对比 |
| 思维教案生成器 | 输入主题 → 自动生成45分钟含思维训练环节的完整教案 |
| 家长周报 | 每周自动生成孩子的思维简报 + 给家长的具体建议 |

## 3.3 B端企业：AI能力建设 + 知识资产管理

| 层级 | 模块 | 功能 |
|------|------|------|
| L1 | 数据安全沙盘 | 15个真实场景代入式推演 |
| L2 | Prompt工程训练 | 实时AI交互反馈 + 并排对比 |
| L3 | AI判断力训练 | 四阶段流程（阅读→找错→揭示→反思） |
| L4 | 人机协作工作流 | 拓扑排序执行引擎 + 3个预置模板 |
| L5 | 组织认知OS | 决策追溯 + 知识流动 + 认知偏差诊断 |
| - | 岗位知识克隆 | 一键捕获关键岗位的全部知识资产 → 即时克隆到新员工 |
| - | AI成熟度诊断 | 五维评分 + 优劣势分析 + 12周提升路线图 + ROI预估 |
| - | 机构知识图谱 | 全机构知识资产可视化 + Token资本 vs 人力资本对比 |
| - | 私有评估框架 | 企业自定义业务指标 + AI影响力自动计算 |
| - | 企业能力名片 | 数据驱动的企业能力公开展示页 |

# 五、竞争格局

## 5.1 竞争定位

```
        ↑ 模型能力
        │  OpenAI · Anthropic · DeepSeek
        │        （思见的下游供应商）
        │
        │  ┌──────────────────┐
        │  │     思见          │  ← 认知层（无人区）
        │  │  "你在怎么想？"    │
        │  └──────────────────┘
        │
        │  ChatGPT · Claude.ai · Kimi · 豆包
        │        （"你知道什么？"）
        │
        └──────────────────────────→ 用户价值
```

思见不与LLM厂商竞争——它使用它们的能力来构建更高层级的价值。它是**模型操作系统**，不是**API中转站**。

## 5.2 核心护城河

1. **61条思维线路数据飞轮**：每次对话都在训练和优化思维识别精度
2. **四层意识识别专利潜力**：纯本地引擎，零成本运行
3. **11种框架可视化算法**：每种有独立的SVG布局算法，难以复制
4. **机构知识图谱**：企业客户的切换成本极高（一旦知识被编码成图谱）
5. **模型无关架构**：知识层独立于LLM，不绑定任何供应商
